 Projeto de Psicologia
Plataforma web educativa sobre transtornos mentais, neurológicos e genéticos com um chatbot inteligente chamado Nina.

 Sobre
Site informativo que oferece informações detalhadas sobre TDAH, autismo, depressão, ansiedade e diversos outros transtornos, com um assistente virtual para suporte e orientação.

 Funcionalidades
Login - Autenticação simples
Página inicial - Cards informativos sobre transtornos
Chatbot Nina - Assistente virtual com respostas personalizadas
3 páginas temáticas - Transtornos de fala, mentais e genéticos
Design responsivo - Funciona em mobile, tablet e desktop

 Arquivos Principais
├── login.html / chat.html / index.html
├── transtorno1/2/3.html
├── style.css / chat.css / login.css
├── script.js
└── logo neurodivergente.png

 Como Usar
Abra login.html no navegador
Insira e-mail e senha
Explore as páginas de transtornos ou converse com Nina
 Chatbot Nina
Responde sobre: ansiedade, depressão, TDAH, autismo, bem-estar geral.

Exemplos de perguntas:

"Olá"
"Tenho ansiedade"
"O que é TDAH?"
"Estou triste"

 Tecnologias
HTML5 | CSS3 | JavaScript Vanilla
